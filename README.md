# Monisha V - Portfolio

A modern, high-performance portfolio website built with React, Vite, Framer Motion, and Tailwind CSS.

## 🚀 Features

- **Dark Theme**: Modern dark color scheme with purple and cyan accents
- **Smooth Animations**: Powered by Framer Motion for fluid transitions
- **High Performance**: Lazy loading, code splitting, and optimized builds
- **Responsive Design**: Works seamlessly on all devices
- **Professional Structure**: Clean, maintainable code architecture

## 🛠️ Tech Stack

- **React 19** - UI library
- **Vite** - Fast build tool
- **Framer Motion** - Animation library
- **Tailwind CSS** - Utility-first CSS
- **React Icons** - Icon library

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Customization

### Update Personal Information

1. **Contact Info**: Edit `src/pages/Contact.jsx`
2. **Skills**: Modify `src/data/skills.js`
3. **Experience**: Update `src/data/experience.js`
4. **Projects**: Edit `src/data/projects.js`

### Change Colors

Update the gradient colors in components:
- Purple: `#9333ea`
- Cyan: `#06b6d4`
- Pink: `#ec4899`
